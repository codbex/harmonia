# Display

Utilities for controlling the opacity of an element.

## Class names

| Class       | Description      |
| ----------- | ---------------- |
| opacity-0   | `opacity: 0%;`   |
| opacity-25  | `opacity: 25%;`  |
| opacity-50  | `opacity: 50%;`  |
| opacity-75  | `opacity: 75%;`  |
| opacity-100 | `opacity: 100%;` |
